# 郑州大学慧灵顿学院官方网站

## 项目简介

这是一个为郑州大学慧灵顿学院设计的官方网站项目，采用现代化的前端技术栈构建，提供学院信息展示、新闻发布、专业介绍、合作交流等功能模块。

## 项目特点

- **现代化设计** - 响应式布局，适配各种设备尺寸
- **单页面应用** - 使用原生JavaScript实现SPA路由系统
- **移动端友好** - 完善的移动端适配和触摸体验
- **双语支持** - 中英文切换功能
- **模块化开发** - 清晰的代码结构和模块化设计

## 技术栈

- **HTML5** - 语义化标签结构
- **CSS3** - 现代化样式设计，包含Flexbox和Grid布局
- **原生JavaScript** - 无框架依赖，轻量级实现
- **ES6+** - 现代JavaScript语法特性

## 核心功能模块

### 1. 导航系统
- 多级下拉菜单设计
- 清晰的栏目分类（学院概况、师资队伍、教学科研等）
- 双语切换功能

### 2. 首页模块
- **轮播图系统** - 自动轮播，支持触摸滑动
- **学院新闻** - 动态加载新闻内容
- **通知公告** - 重要信息发布
- **教学科研** - 展示学术成果
- **专业介绍** - 三个重点专业展示
- **合作交流** - 国内外合作项目

### 3. 路由系统
- 基于Hash的单页面路由
- 动态页面加载
- 浏览器历史记录支持
- 参数化路由处理

### 4. 响应式设计
- 移动端优先设计原则
- 断点系统：768px（平板）、1024px（桌面）
- 弹性布局和媒体查询

## 开发指南

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 本地Web服务器（推荐Live Server）

### 开发建议
1. 使用模块化开发方式，保持代码清晰
2. 遵循CSS命名规范，使用BEM命名法
3. JavaScript采用ES6模块导入导出
4. 图片资源进行适当压缩优化

### 部署说明
1. 将所有文件上传至Web服务器
2. 确保服务器支持静态文件访问
3. 配置适当的缓存策略
4. 建议使用CDN加速静态资源

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+
- 移动端浏览器

## 性能优化

- 图片懒加载
- CSS和JavaScript文件压缩
- 合理的缓存策略
- 减少HTTP请求次数
